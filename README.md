# UnicornXMedia - Premier PR, Events, Podcasts & News Platform

A modern, white-based website featuring PR services, events, podcasts, and news coverage for founders and brands.

## 🎨 Design Features

### Color Scheme
- **Primary**: Deep Blue (#1e40af) - Professional and trustworthy
- **Secondary**: Purple (#7c3aed) - Creative and innovative
- **Accent**: Amber (#f59e0b) - Energetic and attention-grabbing
- **Background**: White with subtle gray tones for a clean, professional look

### Key Features

1. **Custom Cursor**
   - Interactive custom pointer with smooth animations
   - Hover effects that expand the cursor outline
   - Enhances user engagement and modern feel

2. **Smooth Animations**
   - Scroll-reveal animations for content sections
   - Smooth transitions on hover states
   - Counter animations for statistics
   - News ticker with continuous scroll

3. **Responsive Design**
   - Fully responsive across all devices
   - Mobile-first approach
   - Optimized for tablets and desktops

4. **Interactive Tabs**
   - Service sections with tab navigation
   - Smooth content switching
   - PR, Events, Podcasts, and News categories

## � Customization

To customize colors, modify the Tailwind config in the `<script>` section:

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                primary: '#1e40af',    // Change primary color
                secondary: '#7c3aed',  // Change secondary color
                accent: '#f59e0b',     // Change accent color
            }
        }
    }
}
```
